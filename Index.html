⚡ QUY TRÌNH QUẢN LÝ CHUYÊN SÂU

Hệ thống Độc lập - Không xung đột dữ liệu

📂
1. TẠO MẪU SOP ĐỘC LẬP

Mỗi file HTML giờ đây là một "cuốn sổ" riêng. Bạn có thể mở nhiều file (Mã hàng A, Mã hàng B) cùng lúc mà không lo bị đè dữ liệu lên nhau.

✍️
2. THIẾT KẾ & SOẠN THẢO

Mở ⚙️ > Chế độ chỉnh sửa. Dùng Toolbar [A] để làm nổi bật thông số quan trọng (In đậm, đổi màu). Dùng (STEP) trong ô AI để chia bước nhanh.

📥
3. LƯU TRỮ CỨNG (QUAN TRỌNG)

Hệ thống không dùng bộ nhớ tạm trình duyệt. Sau khi soạn xong, bạn PHẢI bấm "LƯU FILE HTML". File tải về sẽ chứa vĩnh viễn nội dung bạn vừa nhập.

🔍
4. KIỂM TRA HIỆN TRƯỜNG

Khi đi kiểm tra, chỉ cần mở file đã lưu. Web luôn ở chế độ "Khóa" để bạn bấm PASS/NG nhanh. Màu sắc thẻ sẽ tự đổi giúp nhận diện lỗi ngoại quan tức thì.

📸
5. XUẤT BÁO CÁO "SẠCH"

Khi bấm PDF/Ảnh, hệ thống thông minh tự chuyển về "Chế độ chỉ xem", ẩn mọi nút chức năng dư thừa để file báo cáo chuyên nghiệp nhất.

💡 Mẹo: Hãy lưu file theo tên [MãHàng_NgayThang.html] để quản lý kho dữ liệu IPQC tốt nhất.
